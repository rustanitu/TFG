The ATFGenerator type exposes the following members.

Constructors

  Name Description
ATFGenerator

Methods

  Name Description
~ATFGenerator
Finalizes an instance of the [ATFGenerator] class.
CalculateGradient
Calculates an aproximation of the voxel's gradient, using its first derivatives.
CalculateLaplacian
Calculates an aproximation of the voxel's laplacian, using its second derivatives.
CalculateVolumeDerivatives
Iterates over the volume, calculating the gradient and the laplacian values for each voxel.
ExtractTransferFunction
Extract a transfer function from the volume, based in Kindlmann's paper. This function needs to be called only once, to do all the calculations.
GenerateGradientSummedHistogram
Acumulates the gradient values of the 3D histogram, generating a 2D hitogram of laplacian values per intensity values. This 2D histogram is generated as an PGM image file in the working directory, named "Gradient Summed Histogram".
GenerateHistogram
Generates a 3D histogram which accumulates occurrences of value-gradient-laplacian triplets. The volume derivatives must have been calculated before this call.
GenerateHistogramSlice
Generates a PGM image file with a histogram slice. The image is generated in the working directory and it's named "Histogram Slice v", where 'v' is the value input.
GenerateHistogramSlices
Generates the all the 256 histogram slices, trhought calls to void GenerateHistogramSlice(int v).
GenerateLaplacianSummedHistogram
Acumulates the laplacian values of the 3D histogram, generating a 2D hitogram of gradient values per intensity values. This 2D histogram is generated as an PGM image file in the working directory, named "Laplacian Summed Histogram".
GetBoundaryDistancies
Gets, for each value, the distance to the closest boundary associated to it. This information is extracted from the summed voxel histogram.
GetGradient
Gets an aproximation of the voxel's gradient, using its first derivatives.
GetLaplacian
Gets an aproximation of the voxel's laplacian, using its second derivatives.
GetSigma
Gets the sigma value of the closest boundary associated the specified value.
GetTransferFunction
Gets the transfer function.
Init
It does all the math necessary so information can be extracted from it. The user will not be able to get any kind of information without a successfull call to this function.
