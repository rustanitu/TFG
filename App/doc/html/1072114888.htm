ITransferFunction Fields

The ITransferFunction type exposes the following members.

Fields

  Name Description
m_color
It storages the color values setted by SetValueColor.
m_color_size
The number of color-value associations.
m_has_color
It storages if a color was setted by SetValueColor for each intensity value.
m_path
The transfer function file path.
m_values_size
The number of intensity values whose distance to closest boundary was given.
